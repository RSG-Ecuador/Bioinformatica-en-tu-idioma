# Código de conducta

Este material fue modificada bajo la licencia CC-BY de las guías de [Software Carpentry en español](https://github.com/swcarpentry/git-novice-es/blob/gh-pages/CONDUCT.md).

Los contribuidores y organizadores de este proyecto nos comprometemos a respetar a todos los miembros de nuestra comunidad. Trataremos de siempre garantizar que la participación en este proyecto sea una experiencia libre de acoso para todos, independientemente del nivel de experiencia, género, orientación sexual, discapacidad, apariencia personal, carrera, etnia, edad, religión, u otro factor.

Los ejemplos de comportamiento inaceptable de los participantes incluyen el uso de imágenes o lenguaje sexual,
comentarios despectivos o ataques personales, trolling, acoso público o privado, insultos, u otra conducta no profesional.

Los organizadores de este proyecto tienen el derecho y la responsabilidad de eliminar, editar o rechazar
comentarios, *commits*, código, problemas y otras contribuciones que no estén alineados con nuestro Código de Conducta. Además, los organizadores del proyecto que no sigan el Código de Conducta pueden ser removidos del equipo del proyecto.

Instancias de comportamiento abusivo, acosador o de otra manera inaceptable puede informarse mediante un correo electrónico a iscb.rsg.ecuador@gmail.com o sebasar1245@gmail.com.
